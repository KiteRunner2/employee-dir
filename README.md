# Employee directory application

```
Simple application to display, search and sort catalogue of users.

Main technologies used:
1. React
2. Bootstrap
```

## Structure of React components

```
React component structure and functions:
 -- App
    -- Header - displays top header on page
    -- SearchPage - loading data from api call, search and sort
        -- TableRow - display results in table format
```

## Features

```
 - User can search directory by typing in search field and results will be updated immediately
 - Search is case insensitive
 - Search is working for name, username, email and id
 - User table can be sorted in ascending or descending order based on name, username,email or id
 - Search results can be sorted
```
